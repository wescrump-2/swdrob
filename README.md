# Savage Owlbear Dice

A custom 3D dice roller extension for Savage Worlds Adventure Edition (SWADE) fans using Owlbear Rodeo. This extension simulates the unique dice mechanics of SWADE, including exploding dice (acing), wild dice, success/raise calculations, and more.

## Installation

To install this extension in Owlbear Rodeo:

1. Open your Owlbear Rodeo campaign
2. Click on the Extensions button in the toolbar
3. Click "Add Extension"
4. Enter the manifest URL: `https://swdrob.vercel.app/manifest.json`
5. Alternatively, you can download the `manifest.json` file and drag it into the Extensions panel
